# Відео файли для фонів

Помістіть ваші відео файли в цю папку:

- `bg-hero.mp4` - відео для головної секції (Hero)
- `bg-exp.mp4` - відео для секції досвіду (Experience)

## Рекомендації:

- **Формат**: MP4 (H.264)
- **Розмір**: 1920x1080 або вище
- **Тривалість**: 10-30 секунд (зациклене)
- **Розмір файлу**: оптимізований для веб (< 10MB на файл)
- **Налаштування**: без звуку, плавне зацикювання

## Автоматичне перемикання:

SmartBackground компонент автоматично:
- ✅ Використовує відео, якщо файл присутній
- ✅ Переключається на зображення, якщо відео не знайдено
- ✅ Показує індикатор завантаження
- ✅ Обробляє помилки завантаження

## Оптимізація відео:

Використовуйте наш скрипт для оптимізації:
```bash
./optimize-media.sh
```

Або вручну через FFmpeg:
```bash
ffmpeg -i input.mp4 -c:v libx264 -crf 28 -preset slow -vf "scale=1920:1080" -an -movflags +faststart output.mp4
```
